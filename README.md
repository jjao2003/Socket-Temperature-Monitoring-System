# Socket Temperature Monitoring System

## Description

This project demonstrates TCP socket communication using Python.

A client sends random temperature data to a server every 5 seconds.

The server receives the data and responds with a confirmation message.

## Files

server.py → TCP server  
client.py → temperature client  

## How to Run

Start server:

python server.py

Start client:

python client.py

## Test Results

Client sends random temperature every 5 seconds.

Example:

TEMP:24.22  
TEMP:26.10  
TEMP:23.88  

Server receives data successfully.
